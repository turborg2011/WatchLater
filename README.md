# WatchLater
Watch Later - iOS app to save films using Kinopoisk API

При реализации использовались библиотека Alamofire для работы с сетью, Kingisher для подгрузки изображений и SnapKit для более удобной верстки кодом. В основном эти библиотеки используются здесь для знакомства с ними. Архитектура - VIPER просто, чтобы изучить ее - очевидно для такого размера проекта она избыточна. Для хранения данных используется CoreData.
